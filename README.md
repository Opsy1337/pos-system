# 🛒 نظام الكاشير POS

نظام نقاط بيع متكامل بواجهة عربية RTL حديثة.

## ⚡ التشغيل السريع

### 1. تشغيل الباكند (Python)

```bash
cd backend
pip install -r requirements.txt
python -m uvicorn main:app --reload --port 8000
```

### 2. تشغيل الفرونتند (Node.js)

```bash
cd frontend
npm install
npm run dev
```

### 3. فتح المتصفح
- **الواجهة:** http://localhost:5173
- **API Docs:** http://localhost:8000/docs

## ✨ الميزات

### شاشة الكاشير
- عرض المنتجات في بطاقات مع الصورة والسعر
- تصفية بالتصنيف + بحث فوري
- سلة مشتريات مع تعديل الكميات
- خصم (مبلغ ثابت أو نسبة %)
- ضريبة القيمة المضافة (15%)
- 3 طرق دفع: نقدي / شبكة / تحويل
- حساب الباقي للدفع النقدي
- طباعة إيصال مناسب لطابعات الكاشير

### لوحة الإدارة
- 📊 **داشبورد** - إحصائيات وأعلى المنتجات مبيعاً
- 🏷️ **المنتجات** - إضافة/تعديل/حذف مع رفع الصور
- 📂 **التصنيفات** - إدارة التصنيفات بالألوان والأيقونات
- 📄 **المبيعات** - سجل كامل مع فلترة بالتاريخ وطريقة الدفع
- 📈 **التقارير** - مخططات يومية/شهرية + أكثر المنتجات مبيعاً
- 📦 **المخزون** - تتبع الكميات وتنبيهات النفاد
- 👥 **المستخدمين** - إدارة الكاشيرين والمديرين
- ⚙️ **الإعدادات** - بيانات المتجر ونسبة الضريبة
